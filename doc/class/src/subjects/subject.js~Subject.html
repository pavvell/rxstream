    Subject

    Direct Subclass:

    BehaviorSubject, ReplaySubject

    Subject is a stream of values, that were pushed into it.

    Example:

    import { Subject } from 'rxstream';

let stream$ = new Subject();
stream$.subscribe(value => console.log(value));
stream$.next(1); // 1
stream$.next(2); // 2
